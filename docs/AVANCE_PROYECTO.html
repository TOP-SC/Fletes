Camino a finalización — prioridades

Acciones ordenadas por impacto en operación.

  1. Import + uso diario prefactura Clickpack + ejecutar macheo +12–15%
  2. Import Tango SA / Limansky (depósito 12, pedidos 102xxxxx) +8–10%
  3. Km reales en rutina — botón batch 500/1000 tras cada import Tango +5–8%
  4. Mejorar macheo fleteros (destinatario / remito) +3–5%
  5. M2: prefactura Gama/Blast +6%
  6. M3: duplicados interior +5% dev / +3% ops
Si hacés solo 3 cosas esta semana: (1) Importar prefactura + macheo, (2) Importar Tango SA, (3) Correr Km reales (1000) en Fletes — la operación podría subir de 62% → 75–78%.
