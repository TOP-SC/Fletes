Camino a finalización — prioridades

Acciones ordenadas por impacto en operación (sin código nuevo).

  1. Reaplicar reglas y proveedores (transporte→proveedor, alertas operativas) +5–8% operación
  2. Import + uso diario prefactura Clickpack + ejecutar macheo +12–15%
  3. Import Tango SA / Limansky (depósito 12, pedidos 102xxxxx) +8–10%
  4. Mejorar macheo fleteros (destinatario / remito) +3–5%
  5. Export maestro Excel pulido (pendiente) +5%
  6. M2: Tango con sucursal + prefactura Gama/Blast +8%
  7. M3: duplicados interior +5% dev / +3% ops
Si hacés solo 3 cosas esta semana: (1) Reaplicar reglas, (2) Importar prefactura + macheo, (3) Importar Tango SA — la operación podría subir de 58% → 73–76%.
