1. ORIGEN DEL PROYECTO
================================================================================

Objetivo inicial:
  App de control de fletes para SommierCenter/Wamaro.
  - Frontend: Streamlit (ágil, práctico).
  - Backend: FastAPI.
  - Fuente principal: Excel Exportacion.xlsx desde Tango (tablero del cliente).
  - Premisa: cada import NO pisa datos anteriores (acumulativo / deduplicado).

Alcance funcional pedido:
  - Automatizar filtros, colores y reglas que hoy hace el operador a mano.
  - Generar maestro de costos (columnas de control con color distinto al Tango).
  - Normalizar valores monetarios (ceros de más: 240000.000000 → $240.000).
  - Tres “mundos” logísticos: Interior (M1), Fletes AMBA/GBA (M2), Duplicados (M3).

Decisiones tempranas:
  - Terminar Mundo 1 antes de seguir.
  - Tarifarios: carpeta data/tarifarios/ + import desde Configuración.
  - Botón reset/vaciar BD para cierre de mes.
  - Iniciar con .bat de un solo doble clic (API + Streamlit).

4. MUNDO 2 — FLETES AMBA/GBA (KM / SUCURSAL / ZONA)
================================================================================

Objetivo:
  Calcular km desde domicilio exacto del detalle Tango (no casos puntuales).
  Lógica general con cache de geocodificación.

Pipeline implementado (fletes_km_service.py):
  1. Cache por remito oficial.
  2. Cache por domicilio_fp (hash domicilio).
  3. Geocodificación + haversine desde sucursal/CD.
  4. Preview estimado por localidad si falla geo.
  5. Reuso de domicilio entre remitos.
  6. Claves pedido-{nro} y fp-{hash} para casos sin remito.

Integración app-wide:
  - Grilla Fletes, detalle auto-calc, cobro maestro, stats, fletes internos.
  - Endpoints POST /fletes/calcular-km y /fletes/caso/{id}/calcular-km.
  - GET /fletes/casos con enrich automático (hasta 30 km por carga).

Limpieza domicilios:
  - Quitar PRINCIPAL:, CASA:, TRABAJO:, etc.
  - Variante caba_domicilio para localidad genérica Buenos Aires/CABA.

Alertas transporte:
  - Transporte 51 vs 40; remito CD solo donde aplica.
  - Una luz por columna en grilla.

Métricas (auditoría jun 2026):
  - 1.640 km reales en cache; 3.337 filas totales flete_distancias.
  - Grilla Fletes: 1.327 casos; 1.101 (83%) con sucursal+km+total; 0 alertas.
  - 129 domicilios AMBA sin geocode (3er batch opcional).
